# AZ Q Pools — Premium Site

Trades Industrial aesthetic for Arizona Quality Pool Service & Repair, LLC.

## Stack
Next.js 14 + Tailwind v4 + GSAP + Lenis + Oswald (display) + Inter (body)

## Brand
- **Palette**: industrial dark `#0F1419` · steel blue accent `#1E90C4` · safety yellow `#FFD93D`
- **Tagline**: "We Clean the Dirty & Fix the Broken"
- **Signature motion**: scroll-driven counters on "By the Numbers" section

## Run
```
npm install
npm run dev
```

## Deploy
Push to GitHub → Vercel import → add `azq-pools.aydenbuilds.com` in Settings → Domains.
